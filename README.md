# W.H.A.T - Walker Harolds Agency Test
WHAT (Walker Harold Agency Test) is an interactive decision-making game where every choice and decision leads to multiple alternative outcomes. You are test subject #6, being questioning by authorities of this agency, you are set to met the right choices and end with the same outcome.

# How to access/install pygame:
1. Open the terminal and type "python -- version"
2. Note: That the lastest python version sometimes don't have access to downloading pygame, so pleaase downgrade it to the earlier versions before the lastest version.
3. Checked if you have installed pip (if there is type "pip --version")
4. Type "python -m pip install pygame.
5. If there are multiple errors or problems in the terminal click this youtube video for more other solutions - [here](https://www.youtube.com/watch?v=5uOTEOJ48G8)

Note: The game cannot play if there is no pygame installed.

# How to play:
1. Click start
2. Enter your name
3. And click the buttons, each buttons lead to two different paths but it will lead to one ending.
4. There is no exit while playing the game.
5. Enjoy!
